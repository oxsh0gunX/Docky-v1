**Docky v1** is a simple, lightweight frontend tool built to accelerate the reconnaissance phase of security assessments. Instead of manually typing Google Dorks, you can simply enter your target domain, and Docky will generate a comprehensive list of categorized dorks, ready to run with a single click.

<br>

<h2 style="border-bottom: 2px solid #4B5563; padding-bottom: 5px;">üöÄ Features</h2>

* **Animated UI:** A clean, dark-mode interface with animated elements.
* **Categorized Dorks:** Dorks are grouped by type for efficient searching:
    * Basic Recon & Subdomains
    * Sensitive Files & Info
    * 3rd Party & Code Leaks
    * Common Vulnerability Patterns (XSS, SQLi, LFI, SSRF, IDOR)
    * High-Impact & Config Dorks
    * Outdated & Archived Content
    * File Upload
* **One-Click Search:** Instantly run any dork in a new Google search tab.
* **Zero Installation:** Runs entirely in your browser. No server or setup needed.

<br>

<h2 style="border-bottom: 2px solid #4B5563; padding-bottom: 5px;">üõ†Ô∏è How to Use</h2>

1.  **Download:** Get the `index.html` and `style.css` files.
2.  **Directory:** Place both files in the **same folder**.
3.  **Open:** Open the `index.html` file in any modern web browser.
4.  **Run:**
    * Enter your target domain (e.g., `example.com`) in the input field.
    * Click the **`Run`** button.
    * Click **`Search`** on any dork to open the results in a new tab.
